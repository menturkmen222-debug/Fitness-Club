# 🏋️ REPLIT AI AGENT — FITNESS CLUB PREMIUM DEMO WEBSITE
## White-Label Auto-Personalization System for Ýeňil Web Agency

---

## 🎯 WHAT YOU ARE BUILDING

A **single, stunning fitness club website** that automatically personalizes itself based on URL parameters. This is a portfolio/demo tool for a web agency — when the agency shares a link with a potential client, the client sees a fully personalized website with THEIR club's name already on it.

**The Magic:** `https://yenil.ru/fitness?name=GymCity&city=Ashgabat&color=FF6B00&phone=%2B99312345678`

Opening this URL renders a complete, premium fitness website that says "GymCity", shows "Ashgabat" location, uses orange as the accent color, and displays their phone number — all without any backend or database.

---

## 🛠️ TECH STACK

```
Framework:    Pure HTML + CSS + Vanilla JavaScript (ONE FILE: index.html)
Animations:   GSAP (CDN) + CSS custom animations
Fonts:        Google Fonts CDN
Icons:        Lucide Icons (CDN)
NO BUILD STEP — works directly in browser, deployable to any static host
```

**Why single file?** Because Salam needs to send ONE link. No backend, no database, no build process. The file reads URL params and renders instantly. Vercel/Replit/GitHub Pages all host it for free.

## 📐 URL PARAMETER SYSTEM

The website reads these URL parameters and applies them throughout:

```javascript
const params = new URLSearchParams(window.location.search);

const CLUB = {
  name:        params.get('name')    || 'FitPro Club',           // Club name
  tagline:     params.get('tag')     || 'Güýçli. Sagdyn. Erkin.', // Hero tagline
  city:        params.get('city')    || 'Aşgabat',               // City name
  phone:       params.get('phone')   || '+993 12 34-56-78',       // Phone number
  address:     params.get('addr')    || 'Garaşsyzlyk şaýoly 100', // Address
  color:       params.get('color')   || 'EF4444',                 // Accent color (hex, no #)
  colorDark:   params.get('colord')  || null,                     // Optional darker shade
  instagram:   params.get('ig')      || '',                       // Instagram handle
  telegram:    params.get('tg')      || '',                       // Telegram handle
  established: params.get('est')     || '2019',                   // Founded year
  members:     params.get('members') || '1200+',                  // Member count
  trainers:    params.get('trainers')|| '25+',                    // Trainer count
  logo:        params.get('logo')    || null,                     // Logo URL (optional)
};

// Auto-generate darker shade if colorDark not provided
// Use color as CSS custom property throughout entire site
document.documentElement.style.setProperty('--accent', '#' + CLUB.color);
```

**EVERY piece of text on the site** must pull from this CLUB object. No hardcoded club-specific content.

## 🎨 DESIGN DIRECTION — DARK PREMIUM ATHLETIC

**Aesthetic:** High-end gym meets editorial magazine. Think Nike meets luxury spa. Dark backgrounds, explosive accent colors, cinematic typography, bold geometric shapes.

**Non-negotiable design choices:**
- **Background:** Near-black `#0A0A0B` as base, NOT pure black
- **Surface cards:** `#111114` with `1px solid rgba(255,255,255,0.08)` border
- **Accent:** Pulled from URL `?color=` parameter — vivid, electric
- **Typography display:** `Bebas Neue` — powerful, sporty, cinematic
- **Typography body:** `DM Sans` — clean and readable
- **Typography numbers/stats:** `Oswald` — impactful stats
- **Texture:** Very subtle noise grain overlay on hero (CSS SVG filter)
- **Shapes:** Diagonal cuts between sections (clip-path: polygon)

**Visual signature element:** A large diagonal stripe/slash shape in the accent color appearing throughout as a recurring motif — on cards, behind headings, as decorative dividers.

### On Scroll (Intersection Observer):
```javascript
// Fade up with stagger for any element with class .reveal
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      gsap.from(entry.target, { 
        y: 50, 
        opacity: 0, 
        duration: 0.7, 
        ease: 'power3.out',
        stagger: 0.1
      });
      observer.unobserve(entry.target);
    }
  });
}, { threshold: 0.15 });

document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
```

```javascript
  const start = performance.now();
  const update = (time) => {
    const elapsed = time - start;
    const progress = Math.min(elapsed / duration, 1);
    const eased = 1 - Math.pow(1 - progress, 4); // ease out quart
    element.textContent = Math.floor(eased * target).toLocaleString() + '+';
    if (progress < 1) requestAnimationFrame(update);
    else element.textContent = target.toLocaleString() + '+';
  };
  requestAnimationFrame(update);
}
```
